Data exchange process that is specific to the Arduino Shield slave Queues the values of the received input into the input_variable and queues all the values to be transmitted into output_variable which is then sent out using ethercat.

Parameters
ec_slave_outputsPointer to the start of the outputs for slave in consideration
oloopNo. of output bytes
ec_slave_inputsPointer to the start of the inputs for slave in consideration
iloopNo. of input bytes
