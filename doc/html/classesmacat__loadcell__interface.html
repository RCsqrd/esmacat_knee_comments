Private function- Data exchange process that is specific to the Esmacat Loadcell Interface slave.

Skip the first 6 bytes of the input pointer [hardware dependent - do not change]

Following that every pair of bytes is a 16 bit ADC reading for the corresponding channel

Parameters
ec_slave_outputsPointer to the start of the outputs for slave in consideration
ec_slave_inputsPointer to the start of the inputs for slave in consideration
