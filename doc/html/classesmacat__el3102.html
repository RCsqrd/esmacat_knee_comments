int  oloop,
uint8_t *  ec_slave_inputs,
int  iloop 
)
virtual

Data exchange process that is specific to the EL3102 slave.

Access the first 6 bytes.

Variable 0 status = Byte 0

Variable 1 value = Bytes 2, Byte 1

Variable 2 status = Byte 3

Variable 3 value = Byte 5, Byte 4

Parameters
ec_slave_outputsPointer to the start of the outputs for slave in consideration
oloopNo. of output bytes
ec_slave_inputsPointer to the start of the inputs for slave in consideration
iloopNo. of input bytes
