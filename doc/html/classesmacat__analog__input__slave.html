esmacat_analog_input_slave::esmacat_analog_input_slave ( )

Constructor for the class that holds the initialization.

Intializes the raw ADC readings for all 16 or ESMACAT_ANALOG_INPUT_MAX_CHANNEL_INDEX channels to 0. And the product code and Vendor ID for the slave are set

Sets a default full-scale-range and offset for default type of single-ended input signals in the range of -12V to +12V

Private function - Configures the type and range of ADC channels with error handling.

Further information can be obtained from the MAX1300 ADC datasheet. Vref = 4096 mV The programmable ranges of the ADC are as follows:

Single-ended, 0 to 3 * Vref/2 range

Single-ended, 0 to 3 * Vref

Single-ended, -3 * Vref/4 to 3 * Vref/4

Single-ended, -3 * Vref/2 to 0

Single-ended, -3 * Vref/2 to 3 * Vref/2

Single-ended, -3 * Vref to 0

Single-ended, -3 * Vref to 3 * Vref

Differential, -3 * Vref/2 to 3 * Vref/2

Differential, -3 * Vref to 3 * Vref

Differential, -6 * Vref to 6 * Vref

Parameters

Private function- Data exchange process that is specific to the Esmacat Analog Input slave.

Skip the first 6 bytes of the input pointer [hardware dependent - do not change]

Following that every pair of bytes is a 16 bit ADC reading for the corresponding channel
