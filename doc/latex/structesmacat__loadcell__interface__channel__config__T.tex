\doxysubsection{Detailed Description}
Struct that holds the Esmacat Loadcell Interface A\+DC channel configuration. 

Describes the type of all the channels (single-\/ended or differential), sets the gain for each A\+DC on the slave, and enables/disables the input buffer for each A\+DC on the slave
